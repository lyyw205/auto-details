# 루미에어 스마트 수면등 상세페이지 디자인

## 디자인 개요

AI 기반 스마트 수면등 "루미에어"의 프리미엄 상세페이지입니다. 딥 네이비(#1F2A44)와 웜 앰버(#FFB347) 컬러를 중심으로 한 다크 테마 디자인이며, 수면·밤·휴식의 분위기를 전달합니다. 타겟은 수면 품질에 관심 많은 2030 직장인이며, 신뢰감과 프리미엄 감성을 동시에 전달하는 톤입니다. 전체 18개 섹션으로 구성됩니다.

## 디자인 시스템

- **캔버스**: 860px 고정 너비 (모바일 상세페이지 기준)
- **메인 컬러**: 딥 네이비 #1F2A44 — 신뢰감, 수면, 밤의 느낌
- **액센트 컬러**: 웜 앰버 #FFB347 — 따뜻한 조명, 포인트 강조
- **배경**: 다크 그라데이션 교차 (#111111↔#1A1A1A), 인접 섹션 톤 반복 금지
- **폰트**: Pretendard, 한국어 최적화, letter-spacing tight
- **히어로 타이틀**: 56px Bold / 섹션 타이틀: 40px Bold / 본문: 22px Regular
- **카드**: 글래스모피즘 (반투명 배경 + backdrop-blur + 미세 보더)
- **텍스트 효과**: 이미지 위 텍스트에는 반드시 드롭 섀도우 + 그라데이션 오버레이
- **포인트 요소**: 브랜드 컬러 그라데이션 텍스트, 넘버 배지(둥근 사각형), 체크 리스트
- **구분선**: 섹션 간 미묘한 그라데이션 라인 또는 도트 분리선
